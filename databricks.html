Big Data Processing Pipeline for Photography Competition

I developed a big data processing pipeline to automate the ingestion, processing, and analysis of image metadata using Azure services, Databricks, and Power BI. The pipeline integrates Computer Vision API to extract metadata from images, stores it in Azure Data Lake and Cosmos DB, and visualizes insights using Power BI.

Project Overview & Technologies

Key responsibilities

  • Deployed Azure resources using custom deployment templates.
  • Configured Azure Storage, Cosmos DB, and Data Factory for data processing.
  • Developed and executed Databricks notebooks to process image metadata.
  • Integrated Azure Data Factory to orchestrate data flow and processing.
  • Implemented Computer Vision API for image analysis and metadata extraction.
  • Designed and optimized data pipelines for metadata ingestion and transformation.
  • Created Power BI dashboards for visualizing image metadata insights.

Technologies used

  • Cloud Services: Microsoft Azure (Storage, Data Factory, Cosmos DB, Synapse Analytics)
  • Big Data & Processing: Databricks, Apache Spark
  • Machine Learning: Azure Computer Vision API
  • Data Visualization: Power BI
  • Programming Languages: Python, SQL
  • Storage & Integration: Azure Blob Storage, JSON, SAS Tokens
