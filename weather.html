Weather Dashboard

The Weather Dashboard is a web application that enables users to search for a city and view its current and future weather conditions using the OpenWeatherMap API. Users must replace "YOUR_API_KEY" with their actual API key in the script.js file. The app dynamically updates the HTML and CSS to display weather information and stores search history in localStorage, allowing easy access to previously searched cities. This tool is ideal for travelers who want to see the weather outlook for multiple cities to plan trips accordingly. The dashboard provides form inputs where, upon searching for a city, users are shown the current and future weather conditions for that city, which is also added to the search history. The current weather display includes the city name, date, weather condition icon, temperature, humidity, and wind speed. For future weather conditions, a 5-day forecast shows the date, weather icon, temperature, wind speed, and humidity. Clicking a city in the search history retrieves its current and future weather conditions again. Technologies used in the project include HTML, CSS, JavaScript, and the OpenWeatherMap API.
