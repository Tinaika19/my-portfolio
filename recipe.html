Recipe Search Engine

This group project is a front-end application developed to manage and interact with favorite recipes, leveraging skills from six modules. It features functionalities for adding, editing, and deleting recipes with client-side storage for data persistence. The application boasts a responsive, polished UI and is deployed on GitHub Pages

Click here to try out the Application

Application Development

Technology used

CSS Framework: Tailwind CSS.

Server side API'S:Edamam, The cocktail DB.

Client side storage:Local Storage. sessions.

Version control:Git, GitHub.

features

Interactive interface: Add, edit, and delete recipes with ease.

Custom modals:For user interactions

Persistent data:LocalStorage ensures data retention across sessions.

responsive design:Optimized for mobile and tablet devices

Usage

Adding Recipes: Select options from dropdown menus or enter favorite cocktails.

Favorite recipes:Drag and drop recipes into the favorite sidebar

Deleting recipes:Remove recipes from the favorite sidebar.
