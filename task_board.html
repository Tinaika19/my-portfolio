Task Tracking Board

The task board application is designed to help project teams organize and manage tasks efficiently. It features a dynamic interface where tasks are displayed in columns representing different progress states (Not Yet Started, In Progress, Completed), with each task color-coded for easy identification. Users can add new tasks via a modal dialog, entering details such as title, description, and deadline, which are then saved to localStorage. Tasks can be dragged between columns to update their progress, and these changes persist after refreshing the page. Tasks can also be deleted, with the deletion being permanent even after a refresh. The application utilizes HTML, CSS, jQuery for dynamic updates, and Day.js for handling dates.
